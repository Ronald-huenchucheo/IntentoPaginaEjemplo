Cambia el punto de mira Si deseas mejorar la puntería o aiming en Counter-Strike: Global Offensive, es importante que sepas usar la consola. A través de esta herramienta, introducirás comandos que te ayudarán a realizar los ajustes que consideres pertinentes. Así, también podrías mejorar tu rendimiento. ¿No sientes comodidad a la hora de usar la mira predeterminada? No hay problema, cambiar el punto de mira en Counter-Strike: Global Offensive es sencillo. Mediante el comando cl_crosshairsize X puedes ajustar el tamaño de la mira. La X debe ser reemplazada con un número entre 0 y 100. ¿Casi no ves el punto de mira o más bien interfiere un poco la visualización? Entonces ajusta su transparencia o solidez con el comando cl_crosshairalpha X. En este caso, la X se reemplaza por un número del 0 al 255. Si quieres que tu mira tenga un color diferente, entonces el comando cl_crosshaircolor X es para ti. Reemplaza la X con números del 0 al 4.
